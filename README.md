# iqbwl.web.id
Personal blog built with **Jekyll** and **Tailwind CSS**.

## Quick Start

**1. Install Dependencies**
```bash
bundle install && npm install
```

**2. Development Server**
```bash
./serve_dev.sh
```
*Runs at `http://localhost:5000` with live reload.*

**3. Production Build**
```bash
./serve_prod.sh
```

## Content Guide

See [tutorial.md](tutorial.md) for instructions on creating posts, pages, and using video embeds.

## Deployment

Pushes to `master` are automatically built and deployed by Netlify.
