---
title: Cara Pakai Vaultwarden
---

Setelah Vaultwarden berhasil kamu install, amankan, dan sudah punya backup rutin, sekarang waktunya kita bahas cara menggunakannya di berbagai perangkat. Vaultwarden ini fleksibel banget, karena kompatibel penuh dengan semua aplikasi resmi Bitwarden. Jadi kamu bisa memakainya di browser, desktop, dan HP dengan mudah.

***
#### 1. Menggunakan Vaultwarden di Google Chrome

* Buka Chrome Web Store dan cari Bitwarden Password Manager.
	
	Link resmi

* Klik Tambahkan ke Chrome lalu aktifkan.

* Buka ekstensi Bitwarden, lalu masuk ke Settings → Server URL.

	Masukkan alamat Vaultwarden kamu, misalnya:
	```bash
	https://vault.domainkamu.com
	```
* Login dengan email dan master password yang kamu buat di Vaultwarden.
* Selesai! Sekarang kamu bisa menyimpan dan mengisi password langsung dari Chrome.

***
#### 2. Menggunakan Vaultwarden di Mozilla Firefox
* Buka Firefox Add-ons dan cari Bitwarden Password Manager.

	Link resmi

* Klik Add to Firefox dan izinkan pemasangan.
* Sama seperti di Chrome, masuk ke Settings → Server URL dan isi alamat Vaultwarden kamu.
* Login, dan semua password siap dipakai.

***
#### 3. Menggunakan Vaultwarden di Desktop

Bitwarden punya aplikasi desktop resmi yang bisa digunakan untuk mengakses Vaultwarden.

Langkahnya:

* Download aplikasi desktop Bitwarden di https://bitwarden.com/download (tersedia untuk Windows, macOS, dan Linux).
* Setelah terpasang, buka aplikasi dan masuk ke Settings.
* Isi Server URL dengan alamat Vaultwarden kamu.
* Login menggunakan akun yang sama.

***
#### 4. Menggunakan Vaultwarden di Android

* Buka Google Play Store dan cari Bitwarden Password Manager.
	
	Link resmi

* Install aplikasinya.
* Buka aplikasi → masuk ke Settings → Self-hosted environment → isi Server URL dengan alamat Vaultwarden.
* Login menggunakan akun kamu.
* Aktifkan Autofill di pengaturan HP supaya password otomatis terisi saat login aplikasi atau website.

***

**Tips Pemakaian Vaultwarden**

* Gunakan folder atau koleksi untuk mengelompokkan password agar rapi.
* Manfaatkan generator password untuk membuat password unik dan kuat.
* Aktifkan 2FA di layanan penting (email, perbankan, dll) dan simpan kode recovery di Vaultwarden.
* Sesekali cek dan hapus password yang sudah tidak digunakan.
